# Transcritor de videos do youtube
## Sobre o Projeto
**Projeto:** Transcritor de videos

**Problema que resolve:** O programa é basicamente um poupador de tempo. Você só precisa inserir a URL de um vídeo do YouTube e o programa realizará a transcrição dele, permitindo que você localize o tópico desejado sem precisar assistir ao vídeo inteiro.
